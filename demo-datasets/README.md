## How to Use

1. Start the Aurora Vibe Charts application
2. Upload any of these datasets using the file upload component
3. View the automatically generated chart recommendations
4. Try natural language prompts to create custom visualizations

## Data Characteristics

- **Small size**: Easy to understand and visualize
- **Clean data**: Minimal null values, proper formatting
- **Diverse types**: Numeric, categorical, datetime, and boolean columns
- **Real-world scenarios**: Business, education, health, weather, finance
- **Multiple formats**: CSV and JSON to test different parsers
